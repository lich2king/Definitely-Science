Definitely Science

Definitely Science is an unblocked games website that offers everything from Minecraft to Monkey Mart. Definitely Science was founded in 2021 but only found a large user base January of 2022. The goal of Definitely Science is to provide the most entertaining experience to bored students who need a break. We have hundreds of free to play games. Every month over 250 thousand gamers from all over the world play their favorite games on Definitely Science.

Why are unblocked games popular?

Students love unblocked games because they get to play whatever they want, unrestricted and free. Competing with other students and trying to set high scores is a great way to kill some time. All the games on Definitely Science are fast to load and require no download.

What are unblocked games?

Unblocked games are games that bypass filters and block protocols set by network administrators. This implies that gamers may access the games from anywhere, including school, work, and public Wi-Fi networks such as coffee shops or libraries.

Can I deploy DS on my server?

Yes, you can deploy Definitely Science (DS) on your own server. The source code for DS is available on GitHub and can be deployed on any server that meets the necessary requirements. To do so, you'll need to clone the repository from GitHub, ensure your server environment meets the necessary dependencies, and follow the deployment instructions provided in the repository's documentation. By deploying DS on your server, you gain control over the configuration and can customize it to suit your specific needs.

Deploy DS via Terminal:

  1. git clone https://github.com/lich2king/Definitely-Science.git DS
  2. cd DS
  3. npm install
  4. npm start

If you want to add step 4 into pm2:
pm2 start npm --name "DS" -- start
